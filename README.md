# TurkStudent CO - LLM Jetpack Ders Notları

Bu depo, TurkStudent CO tarafından hazırlanan "LLM Jetpack" dersine ait ders notlarını ve örnek Jupyter notebook'ları içerir. Ders materyalleri eğitim amaçlıdır ve model eğitim, ince ayar (fine-tuning) ve örnek kullanım senaryolarını kapsar.

İçerik genel başlıkları:
- Basit LLM eğitim örnekleri
- Model karşılaştırma örnekleri
- Küçük ölçekli fine-tuning örnekleri

Not: Bu notlar eğitim amaçlıdır; gerçek üretim iş yükleri için uygun donanım ve güvenlik önlemleri alınmalıdır.
